# Lessons Learned

失敗・ハマりから学んだことを保管する。同じミスを繰り返さないための記録。

## 対象

- 原因究明に時間がかかったバグや設定ミス
- 予想外の挙動・落とし穴
- 環境構築時の注意点

## ファイル命名規則

`YYYY-MM-DD_{topic}.md` 例: `2026-02-23_junction-pitfall.md`

## フロントマター

```yaml
---
author: ai
created: YYYY-MM-DD
type: lesson-learned
project: {由来プロジェクト名}
tags: [ai-memory, lesson-learned]
---
```

## 記録形式

```markdown
## 問題

何が起きたか

## 原因

なぜ起きたか

## 解決策

どう直したか

## 再発防止

次回どうすれば防げるか
```
