# Project Summary

## 概要

- **プロジェクト名**: GRANDIT ERP AI Integration
- **目的**: GRANDIT ERPに生成AI機能を統合し、仕訳検証・業務プロセス自動化を実現
- **期間**: 2025/10 ～ 2026/06
- **現在のフェーズ**: 基本設計・プロトタイプ実装

## 技術スタック

| カテゴリ | 技術 |
|---------|------|
| 言語 | C#, Python, TypeScript, SQL |
| DB | SQL Server 2019 |
| インフラ | Azure (App Service, SQL Database) |
| AI | Claude API, MCP Server, Dify |

## アーキテクチャ概要

```
[ユーザー] → [Dify Workflow] → [Claude API / MCP Server]
                                       ↓
                                [SQL Server (GRANDIT DB)]
```

## 現在の状態

- **完了済み**: DB接続MCPサーバー構築、仕訳検証プロンプト設計、Dify環境構築
- **進行中**: 管理画面UI/API実装、デモ準備
- **未着手**: 本番データテスト、運用マニュアル
- **ブロッカー**: 本番DB接続のセキュリティレビュー待ち

## 主要な決定事項

| 日付 | 決定内容 | 理由（1行） |
|------|---------|------------|
| 2026-02-10 | Difyをワークフロー基盤に採用 | ノーコードで非エンジニアも活用可能 |

## 注意事項

- 本番DBには読み取り専用接続のみ（INSERT/UPDATE/DELETE禁止）
- SQL Server 2019 の構文に準拠
- GRANDIT固有のテーブル命名規則あり（shared/docs/design/ 参照）
- 3月AI発表会に向けたデモ準備を並行で進める
